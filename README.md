## 🍔 Food Ordering System
A simple online Food Ordering System developed using **PHP, MySQL, HTML, CSS, and JavaScript**. The project allows customers to browse food items, place orders, and contact the restaurant. It also includes an admin panel for managing food items and customer orders.
## 🚀 Live Demo
🌐 Website: https://crispycorner.rf.gd
## 📌 Features
### Customer Side
- Home page
- Food menu
- Food gallery
- Shopping cart
- Checkout system
- User login
- Contact page
- About page
### Admin Panel
- Admin Login
- Add Food Items
- Edit Food Items
- Delete Food Items
- Upload Food Images
- Manage Orders
- Database Management
## 🛠️ Technologies Used
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
